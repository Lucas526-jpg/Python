# Proyectos y practicas en python 🐍

Este repositorio contiene varios scripts de práctica y proyectos en python, organizados por temáticas. Aquí practico conceptos como generación de contraseñas, automatización, manejo de datos (JSON/CSV) y Web Scraping.

## 📂 Indice


### 🔐 Generador de Contraseñas
Scripts ubicados en la carpeta `Generador_contrasenias/`:
* [pass.py](Generador_contrasenias/pass.py): Script encargado de generar contraseñas seguras de forma aleatoria.

### 🛠️ Pruebas y Automatización
Scripts ubicados en la carpeta `Prueba_virt/`:
* [alert.py](Prueba_virt/alert.py): Script de pruebas para manejo de alertas.
* [lectura_json_a_csv.py](Prueba_virt/lectura_json_a_csv.py): Utilidad para leer archivos JSON (como `archivo.json`) y transformarlos a formato CSV.
* [portapapeles.py](Prueba_virt/portapapeles.py): Automatización para leer y escribir en el portapapeles del sistema operativo.
* [snowstorm.py](Prueba_virt/snowstorm.py): Script de nieve cayendo.
* [web_scrap.py](Prueba_virt/web_scrap.py): Proyecto de Web Scraping basico para extraer datos de páginas web.
